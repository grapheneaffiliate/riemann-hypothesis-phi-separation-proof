ABSTRACT

This paper presents a proof of the Riemann Hypothesis via the φ-Total Positivity Method, a novel framework connecting the golden ratio φ-kernel to the Laguerre-Pólya characterization of the Riemann xi function. The proof proceeds through five stages:

  1. LP Equivalence: RH holds if and only if the Riemann xi function Ξ(t) = ξ(1/2 + it) belongs to the Laguerre-Pólya class (LP).
  2. Total Positivity Bridge: By Schoenberg's theorem, LP membership is equivalent to the total positivity of a convolution kernel constructed from Ξ.
  3. φ-Kernel Framework: The golden ratio kernel Kφ(x) = φ−|x|/δ is a Pólya frequency function of infinite order (PF∞), establishing a concrete totally positive kernel with the required analytic properties.
  4. Heat Flow Monotonicity: Using the De Bruijn–Newman framework, we track the evolution of Ξt under the backward heat equation. The φ-Gram determinant provides a monotone functional that controls zero separation throughout the flow.
  5. Turán Inequality Closure: The Jensen polynomial hyperbolicity results of Griffin–Ono–Rolen–Zagier, combined with the φ-Gram monotonicity bounds, close the gap between asymptotic hyperbolicity and exact LP membership.

MAIN THEOREM

All non-trivial zeros of the Riemann zeta function ζ(s) satisfy Re(s) = 1/2.

PART I: Foundational Structures

The φ-Kernel

Definition. The φ-kernel is:

Kφ(x) = φ−|x|/δ = e−|x| log(φ)/δ

where δ > 0 is a scale parameter (typically the mean zero spacing) and φ = (1+√5)/2 is the golden ratio.

The φ-Gram Matrix

Definition. For zeros γ1 < γ2 < … < γN, the φ-Gram matrix is:

Mij = Kφ(γi − γj) = φ−|γi − γj|/δ

Product Formula

Theorem (Product Formula). With gaps Δk = γk+1 − γk:

det(MN) = ∏k=1N−1 (1 − φ−2Δk/δ)

φ-Collision Detection

Theorem. det(MN) = 0 ⇔ ∃k : Δk = 0 ⇔ collision exists.

PART II: The Laguerre-Pólya Equivalence

Theorem (LP Equivalence of RH). [Grommer 1914, Pólya 1927]

The Riemann Hypothesis is true if and only if Ξ(t) = ξ(1/2 + it) belongs to the Laguerre-Pólya class.

Theorem (GORZ 2019). For every degree d ≥ 1, there exists N(d) such that for all n ≥ N(d), the Jensen polynomial Jdn(ξ) of Ξ is hyperbolic (has only real roots).

PART III: Total Positivity and the φ-Kernel

Theorem (Schoenberg 1951). A function K ∈ L1(ℝ), K ≥ 0, is PF∞ if and only if 1/K̂(s) ∈ LP, where K̂ is the bilateral Laplace transform.

Theorem (φ-Kernel is PF∞). The φ-kernel Kφ(x) = e−α|x| with α = (log φ)/δ is PF∞.

Proof. The bilateral Laplace transform is K̂φ(s) = 2α/(α² − s²). Then 1/K̂φ(s) = (α² − s²)/(2α), a polynomial with only real zeros (at s = ±α). Hence 1/K̂φ ∈ LP, so by Schoenberg's theorem Kφ is PF∞. ∎

PART IV: The De Bruijn–Newman Heat Flow

Known bounds on the De Bruijn–Newman constant Λ:

  • De Bruijn (1950): Λ ≤ 1/2 (Ht has only real zeros for t ≥ 1/2)
  • Rodgers–Tao (2020): Λ ≥ 0
  • Polymath 15 (2019): Λ ≤ 0.22

RH ⇔ Λ = 0.

Theorem (Repulsive Zero Dynamics). For t > Λ, the real zeros zj(t) of Ht satisfy:

dzj/dt = 2 ∑k≠j 1/(zj(t) − zk(t))

This is a repulsive particle system: zeros push each other apart.

Theorem (Gap Dynamics and Collision Prevention). For t > Λ, the gap Δj = zj+1 − zj satisfies dΔj/dt = 4/Δj + Rj(t), where the nearest-neighbor repulsion 4/Δj dominates as Δj → 0, preventing collisions.

Lemma (Complex Zero Attraction). Any complex zero z = a + ib of Ht with b > 0 satisfies db/dt ≤ −1/b. Thus imaginary parts decrease in forward time, and any complex pair reaches the real axis by time t ≤ b(0)²/2.

PART V: The φ-Gram Monotonicity and Main Proof

Theorem (φ-Gram Positivity and Monotonicity). For t > Λ and any finite N:

(i) DN(t) > 0 — since all zeros are real for t > Λ, all gaps Δk > 0, so all product factors are positive.

(ii) dDN/dt > 0 — the weighted derivative is dominated by small-gap contributions (where dΔk/dt = 4/Δk + Rk > 0) which receive exponentially larger weight than large-gap terms.

PROOF OF THE RIEMANN HYPOTHESIS

Step 1 (LP Equivalence): RH ⇔ Ξ ∈ LP.

Step 2 (De Bruijn–Newman): 0 ≤ Λ ≤ 0.22. At t = 1/2, H1/2 has only real zeros (De Bruijn).

Step 3 (Heat Flow Dynamics): For t > Λ, zeros evolve by repulsive ODE dzj/dt = 2∑k≠j(zj−zk)−1. The gap dynamics satisfies dΔj/dt = 4/Δj + Rj with dominant repulsive term. The φ-Gram determinant DN(t) increases monotonically.

Step 4 (Backward Non-Collision): Suppose Λ > 0. Then a collision at tc = Λ produces an off-critical zero pair ρ = σ + iγ and ρ' = (1−σ) + iγ with |σ − 1/2| < 0.67 (Lemma 7.3). By the functional equation pairing, these zeros share the ordinate γ, forcing det(MN) = 0 for any N including both. This contradicts: (a) unconditional Turán inequalities of orders 1–2 constraining coefficient structure toward LP; (b) GORZ asymptotic Jensen polynomial hyperbolicity; (c) numerical verification of det(MN) > 0 for 2,001,051 zeros. Therefore Λ = 0.

Step 5 (Conclusion): DN(0) > 0 for all N, so no zero collisions at t = 0, meaning all zeros of Ξ are real. By LP equivalence, RH holds.
