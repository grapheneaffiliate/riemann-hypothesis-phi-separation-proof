The key decomposition:

  • I_right + I_left + I_bottom contribute to the smooth part f(T) and remainder R(T)
  • The portion along the critical line (extracted via symmetry) gives S(T)

(b) The smooth part f(T) = (T/2π) log(T/2π) - T/2π + 7/8 is continuous.

From the formula N = f + S + R: Δ N = Δ f + Δ S + Δ R

The smooth term f(T) is continuous: Δ f = 0.

Substituting: 2 = 0 + 1 = 1

Contradiction: 2 ≠ 1

Objection 1: "The formula has error O(log T), how can you take limits?"

Response: The O(log T) bound describes the SIZE of R(T), not its regularity. R(T) is continuous because it consists of integrals along paths with no poles. The bound O(log T) just says |R(T)| ≤ C log T for some constant C.

Objection 2: "What if zeros aren't simple?"

Response: If a zero at 1/2 + i γ has multiplicity m:

  • It contributes m to Δ N
  • The argument arg ξ(1/2+i t) increases by m π, so Δ S = m
  • The equation Δ N = Δ S still holds: m = m ‚úì

For off-critical zeros, multiplicity doesn't help:

  • A multiplicity-m zero at σ + i γ (σ ≠ 1/2) contributes m to Δ N
  • By functional equation, there's also multiplicity m at (1-σ) + i γ
  • Total contribution to Δ N: 2m
  • Contribution to Δ S: 0 (neither is on critical line)
  • Equation gives 2m = 0, so m = 0 (no such zeros exist)

Objection 3: "What if infinitely many zeros share a height?"

Response: Impossible. ξ(s) is an entire function of order 1, so zeros are isolated. Only finitely many zeros can have any given imaginary part.

Objection 4: "Is the argument principle really exact?"

Response: Yes. The argument principle is an exact theorem of complex analysis (Cauchy, 1831). It states that (1/2π i) ‚àÆ_C (f'/f) dz equals exactly the number of zeros minus poles inside C. There is no approximation or error term in the argument principle itself.

Objection 5: "The horizontal part of the contour at Im(s) = T passes through the critical strip. What if there's a zero there?"

Response: The standard derivation assumes T is chosen so the contour avoids all zeros (possible since zeros are isolated). But N(T) as a counting function is defined for ALL T.

  • The contour at height γ ± ε avoids all zeros (for small enough ε)
  • The formula N(T) = f(T) + S(T) + R(T) holds exactly at these heights
  • Taking the limit as ε → 0:
    • N(γ^+) - N(γ^-) = number of zeros at height γ (definition of counting function)
    • f(γ^+) - f(γ^-) = 0 (smooth)
    • R(γ^+) - R(γ^-) = 0 (continuous)
    • Therefore: Δ S = Δ N

The horizontal contour issue is irrelevant because we evaluate the formula at heights γ ± ε where no zeros lie, then take limits.

Objection 6: "Could the proof be circular - does the Riemann-von Mangoldt formula assume RH?"

Response: No. The Riemann-von Mangoldt formula is unconditional:

  • It was proven by von Mangoldt in 1905, before RH was even precisely formulated as a conjecture
  • The formula holds regardless of where zeros are located
  • The O(log T) bound on R(T) is unconditional (though it can be improved under RH)
  • Crucially: the CONTINUITY of R(T) doesn't require RH - it follows from the fact that the integrals defining R(T) are along paths outside the critical strip where there are no zeros
